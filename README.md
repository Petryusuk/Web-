# Веб-проєкт: графік функції

Це простий веб-проєкт із файлами `index.html`, `style.css`, `script.js`.

## Запуск локально
Відкрийте `index.html` у браузері або використайте легкий HTTP-сервер.

## Структура
- `index.html` — розмітка
- `style.css` — стилі
- `script.js` — логіка на JS
